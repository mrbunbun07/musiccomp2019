What is Audulus 3?

Audulus 3 is a sound design and music creation tool that uses modular elements to synthesize various sounds. These sounds are made by chaining devices called nodes in various ways. By using these nodes, one can generate and modify sounds at will.

LEARN MORE

The Plan

Ever since I saw the magical capabilities of modular synthesis, I wanted to learn more about it. However due to financial and physical limitations, I opted to purchase Audulus 3 instead. My individual plan for the class is to learn how modular synthesis works, to learn how various sounds can be synthesized, and to create an electronically-based song using only the sounds I've created.

Learn Audulus

Learn how modular synthesis works, how different modules are patched together, and how certain sounds are made. Learn how to use Audulus, what each node does, where the nodes can be used. Learning the concepts of patches and subpatches.

Experiment

Experiment within Audulus to develop audio manipulation strategies. Familiarize self with the program. Try to build tools that will be helpful in the future. Discover which combinations of nodes make what sounds.

Create Sounds

Create sounds to use in the song. These can range from melodic elements to sound effects. Sounds do not necessarily have to be polished since they will be reviewed during the composistion phase.

Compose Song

Using the sounds created, compose and produce an electronic song in Ableton Live. Make necessary edits to samples and begin an arrangement. Using the sounds provided, lay out the basic structure of the song. Add details using plugins and built-in effects.
