AUDULUS

Individual Project

What is Audulus 3?

Audulus 3 is a sound design and music creation tool that uses modular elements to synthesize various sounds. These sounds are made by chaining devices called nodes in various ways. By using these nodes, one can generate and modify sounds at will.

LEARN MORE

The Plan

Ever since I saw the magical capabilities of modular synthesis, I wanted to learn more about it. However due to financial and physical limitations, I opted to purchase Audulus 3 instead. My individual plan for the class is to learn how modular synthesis works, to learn how various sounds can be synthesized, and to create an electronically-based song using only the sounds I've created.
